Events, Event Bubbling, setInterval & setTimeout

In web development, events are actions that occur in the browser, such as clicks, key presses, or page loads. JavaScript allows you to respond to these events using event listeners.

Event Bubbling

Event bubbling is a mechanism where an event starts from the target element and bubbles up to its parent elements. This allows for event delegation, where a single event listener can manage events for multiple child elements.

setInterval & setTimeout

setInterval() and setTimeout() are functions that allow you to execute code after a specified delay or repeatedly at specified intervals.

setTimeout() executes a function once after a delay:

setTimeout(() => {
        console.log('Executed after 2 seconds');
    }, 2000);

setInterval() executes a function repeatedly at specified intervals:

setInterval(() => {
        console.log('Executed every 3 seconds');
    }, 3000);
